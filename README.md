# HTML5 test

This is my submission to Digia for the front-end position.



### Installation


Clone the git repository.

```
git clone https://github.com/miavii/digia-test.git
```

Open the repository's folder

```
cd digia-test
```
Run the app with
```
yarn start
```
or
```
npm start
```
